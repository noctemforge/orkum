# ORKUM
Low level file inspector for the detail oriented user.

#### Debug
- RAW inspect and edit
- Quick format
- Validation schema for files
- Custom DSL support 

#### PKI 
- Certificate replication
- Quick reissue
- Multiple issue
- Chain copy
